Posições


Posição Estática

Um elemento com position: static não está posicionado de nenhuma maneira especial; ele está sempre posicionado de acordo com o fluxo da página.

Na posição static, os elementos não são afetados pelas propriedades: top, left, right e bottom.

Position Static

Posição Relativa

Um elemento com position: relative é posicionado em relação à sua posição normal.

Nessa posição, os elementos são afetados pelas propriedades: top, left, right e bottom.

O conteúdo não vai se ajustar para se encaixar no espaço deixado pelo elemento.

Position Relative

Posição Fixa

Um elemento com position: fixed é posicionado em relação à janela de visualização. Mesmo que você role a página, o elemento permanece fixo no mesmo lugar.

Position Fixed

Posição Absoluta

Um elemento com position: absolute é posicionado em relação ao seu ancestral mais próximo que não tenha a posição static. Se não houver tal ancestral, é posicionado em relação ao elemento html.

Ancestral
Position Absolute

Posição Sticky

Um elemento com position: sticky se comporta como relative até que atinja um determinado ponto de rolagem, quando então se comporta como fixed.

Position Sticky
